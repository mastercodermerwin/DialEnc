
# DialEnc 1.0.0

DialEnc is a encryption based on dialing system in phones.



## Usage/Examples

```python
    dialenc = DialEnc()
```

```python
    enc_data = dialenc.encrypt("hello")
    print(f"Encrypted :  {enc_data}")

```

```python
    dec_data = dialenc.decrypt(enc_data)
    print(f"Decrypted :  {dec_data}")
```
